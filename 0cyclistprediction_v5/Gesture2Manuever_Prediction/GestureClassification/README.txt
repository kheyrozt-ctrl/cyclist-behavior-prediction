GestureClassification 存放 Stage1 gesture/action 分类器。
GestureClassification stores the Stage1 gesture/action classifiers.

Stage1 包含三个分支：头部、上肢和腿部。
Stage1 contains three branches: head, upper-limb, and leg.

主入口文件是 `main.py`。
The main entry file is `main.py`.

data loader 读取 `../VideoCrop/GestureCrop_newsplit.py` 生成的 pkl 文件。
The data loader reads the pkl files generated by `../VideoCrop/GestureCrop_newsplit.py`.

模型代码定义每个分支使用的 BiLSTM 分类器。
The model code defines the BiLSTM classifier used by each branch.
